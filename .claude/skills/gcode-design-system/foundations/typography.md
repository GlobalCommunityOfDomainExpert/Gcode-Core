# Typography

## Purpose

Typography creates a clear visual hierarchy and ensures consistent readability across GCODE.

---

## Font Family

### Primary

**Inter**

Used for:

- Headings
- Body text
- Buttons
- Labels
- Navigation
- Forms

### Monospace

**JetBrains Mono**

Used only for:

- Booking references
- IDs
- Route paths
- Code snippets
- System values

---

## Typography Scale

| Style | Use |
|--------|-----|
| Small | Captions, helper text, badges |
| Body | Paragraphs, inputs, buttons |
| Large | Card titles, dialogs |
| Heading | Section headings |
| Display | Hero titles, major page headings |

---

## Weight Guidelines

| Weight | Use |
|---------|-----|
| Regular | Body text |
| Medium | Labels |
| Semibold | Navigation |
| Bold | Buttons |
| Extra Bold | Titles |

---

## Rules

- Use Inter for all UI text.
- Use JetBrains Mono only for technical information.
- Keep typography consistent across pages.
- Use larger text to create hierarchy instead of multiple colors.
- Avoid using more than three font sizes within the same section.
- Maintain comfortable line spacing for paragraphs.
