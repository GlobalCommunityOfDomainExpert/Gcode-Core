# Tabs

## Purpose

Allows users to switch between different views, categories, or filters within the same context.

---

## Composition

Multiple Button/Link atoms arranged horizontally.

---

## States

Active

Inactive

Hover

---

## Responsive

Scrolls horizontally on small screens.

---

## Uses

→ atoms/button.md

---

## Example

```html
<div class="g-tabs">
  <button class="g-tabs__item g-tabs__item--active">All</button>
  <button class="g-tabs__item">Events</button>
  <button class="g-tabs__item">Webinars</button>
</div>
```
